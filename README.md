# Recipe Recommended Web Project (Internship Test)

โปรเจกต์นี้เป็นส่วนหนึ่งของแบบทดสอบการเข้าฝึกงาน โดยพัฒนาหน้าเว็บ Responsive สำหรับแสดงรายการอาหารตามโจทย์ที่ได้รับมอบหมาย

* **Frontend Framework:** Vue 3 (Vite)
* **Styling:** Standard CSS 
เพื่อให้โปรเจกต์ทำงานได้ในเครื่องของคุณ กรุณาทำตามขั้นตอนดังนี้:

1. **Clone Repository**
   git clone [ใส่ลิงก์ GitHub ของคุณที่นี่]
   cd my-recipe-app
2. Install Dependencies
    npm install
3. Run Development Server
    npm run dev
